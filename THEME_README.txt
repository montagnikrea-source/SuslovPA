   • Эмодзи меняется с временем суток

✅ Система определения предпочтений
   • Смотрит что установлено в ОС
   • Уважает выбор пользователя
   • Работает на Windows, macOS, Linux, iOS, Android

💾 СОХРАНЕНИЕ ДАННЫХ

Ваш выбор темы сохраняется в localStorage браузера:
  • Локальное хранилище (на вашем устройстве)
  • Не отправляется на сервер
  • Не видно другим пользователям
  • Остается даже после перезагрузки

Как очистить:
  F12 → Application → Local Storage → Удалить ключ:
  • theme-preference
  • theme-set-at

🐛 ОТЛАДКА

Если что-то не работает:

1. Откройте консоль: F12 → Console

2. Проверьте текущую тему:
   document.documentElement.getAttribute('data-theme')

3. Проверьте сохраненное предпочтение:
   localStorage.getItem('theme-preference')

4. Полностью сбросьте:
   localStorage.clear();
   location.reload();
