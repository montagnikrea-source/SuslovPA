🎨 СИСТЕМА ДИНАМИЧЕСКИХ ТЕМ ОФОРМЛЕНИЯ

Сайт теперь имеет профессиональное оформление с автоматическим переключением
между светлой и темной темой в зависимости от времени суток.

═══════════════════════════════════════════════════════════════════════════════

🌟 ОСНОВНЫЕ ВОЗМОЖНОСТИ

1. 🌅 АВТОМАТИЧЕСКОЕ ПЕРЕКЛЮЧЕНИЕ ПО ВРЕМЕНИ СУТОК
   • 06:00-12:00 (Утро)    → ☀️ Светлая тема
   • 12:00-18:00 (День)    → ☀️ Светлая тема  
   • 18:00-21:00 (Вечер)   → 🌙 Темная тема
   • 21:00-06:00 (Ночь)    → 🌙 Темная тема

2. 🔘 РУЧНОЕ ПЕРЕКЛЮЧЕНИЕ
   • Кнопка в топ-баре (справа вверху)
   • Один клик = смена темы
   • Выбор сохраняется навсегда

3. 🕐 ЧАСЫ РЕАЛЬНОГО ВРЕМЕНИ
   • В топ-баре слева
   • Показывает текущее время (HH:MM:SS)
   • Эмодзи меняется с временем суток (🌅 ☀️ 🌆 🌙)

4. 🖥️ ОПРЕДЕЛЕНИЕ СИСТЕМНОГО ПРЕДПОЧТЕНИЯ
   • Автоматически определяет: macOS/Windows/Linux Dark Mode
   • iOS/Android Dark theme
   • Уважает настройки операционной системы

5. 💾 СОХРАНЕНИЕ ВЫБОРА
   • Все сохраняется в localStorage
   • Работает даже без интернета
   • Выбор остается после перезагрузки

🔄 КАК СИСТЕМА РАБОТАЕТ

1. Пользователь открывает сайт

2. JavaScript функция initThemeSystem() запускается

3. Система определяет:
   • Текущее время суток (утро/день/вечер/ночь)
   • Сохраненное пользовательское предпочтение
   • Системное предпочтение ОС

4. Применяется подходящая тема

5. Часы обновляются каждую секунду

6. Пользователь может нажать кнопку переключения

7. Выбор сохраняется в localStorage

8. При следующем визите система помнит выбор
