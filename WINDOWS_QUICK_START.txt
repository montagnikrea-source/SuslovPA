                    БЫСТРЫЕ КОМАНДЫ ДЛЯ WINDOWS (PowerShell)

═══════════════════════════════════════════════════════════════════════════════

🔍 ШАГ 1: Найдите папку SuslovPA на диске C:

Скопируйте и выполните в PowerShell:

    Get-ChildItem -Path C:\ -Filter "SuslovPA" -Recurse -Force -ErrorAction SilentlyContinue | ForEach-Object { $_.FullName }

Это выведет ВСЕ пути, где находится папка SuslovPA.


═══════════════════════════════════════════════════════════════════════════════

🔍 ШАГ 2: Если папка не найдена, клонируйте репозиторий:

Скопируйте и выполните:

    cd C:\Users\Suslo
    git clone https://github.com/montagnikrea-source/SuslovPA.git
    cd SuslovPA

Готово! Репозиторий клонирован.


═══════════════════════════════════════════════════════════════════════════════

✅ ШАГ 3: Загрузите последние изменения и разверните:

Замените "C:\ПОЛНЫЙ\ПУТЬ\К\SUSLOVPA" на реальный путь и выполните:

    cd C:\ПОЛНЫЙ\ПУТЬ\К\SUSLOVPA
    git fetch origin
    git pull origin main
    git pull origin gh-pages
    git checkout main
    git add -A
    git commit -m "Deploy: anti-oscillation damping final"
    git push origin main gh-pages

✅ ШАГ 4: Проверьте развёртывание (через 10 минут):

    curl https://suslovpa.vercel.app/noninput.html | grep -c OscillationDamper
    curl https://montagnikrea-source.github.io/noninput.html | grep -c OscillationDamper

Оба должны вывести число > 0 (обычно 2 или 3)

❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

Проверьте что git настроен:
    git remote -v

Должно быть:
    origin  https://github.com/montagnikrea-source/SuslovPA.git (fetch)
    origin  https://github.com/montagnikrea-source/SuslovPA.git (push)

Если нет, добавьте:
    git remote add origin https://github.com/montagnikrea-source/SuslovPA.git
