Telegram

Проект содержит скрипты для интеграции с Telegram (см. telegram-monitor.js, telegram-debug-script.js). Для связи через Telegram используйте канал/бот, описанный в документации репозитория (Secrets/токены устанавливаются на стороне сервера).

По электронной почте

Адрес электронной почты в коде репозитория не найден. Если вы хотите оставить контактный e‑mail, откройте issue и укажите предпочтительный способ связи — мы добавим его на страницу.

Спасибо за интерес к проекту — все коммуникации лучше вести через GitHub Issues.
